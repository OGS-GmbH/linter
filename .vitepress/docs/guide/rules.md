# Rules

This package supports following pre-configured rulesets:

- eslint
  - ✅ [Globals](https://www.npmjs.com/package/globals)
  - ✅ [JavaScript](https://eslint.org/docs/latest/rules/)
  - ✅ [Markdown](https://github.com/eslint/markdown?tab=readme-ov-file#rules)
  - ✅ [JSON](https://github.com/eslint/json?tab=readme-ov-file#rules)
- [eslint-plugin-jsdoc](https://github.com/gajus/eslint-plugin-jsdoc?tab=readme-ov-file#user-content-eslint-plugin-jsdoc-rules)
- [eslint-plugin-security](https://github.com/eslint-community/eslint-plugin-security?tab=readme-ov-file#rules)
- [eslint-plugin-unicorn](https://github.com/sindresorhus/eslint-plugin-unicorn?tab=readme-ov-file#rules)
- [typescript-eslint](https://typescript-eslint.io/rules/)
- [stylistic](https://eslint.style/rules#rules)
  - ✅ JavaScript
  - ✅ TypeScript
  - ✅ Plus
- [stylelint](https://stylelint.io/user-guide/rules)
- [stylelint-scss](https://github.com/stylelint-scss/stylelint-scss?tab=readme-ov-file#list-of-rules)
- angular-eslint
  - ✅ [Templates](https://github.com/angular-eslint/angular-eslint/blob/main/packages/eslint-plugin-template/README.md)
  - ✅ [TypeScript](https://github.com/angular-eslint/angular-eslint/blob/main/packages/eslint-plugin/README.md)
- [commitlint](https://commitlint.js.org/reference/rules.html)
- [package-json-lint](https://npmpackagejsonlint.org/docs/rules)
